How to Build
------------

First, you should install `spm` and `spm-build`:

```
$ npm install spm -g
$ npm install spm-build -g
```

Then, build it:

```
$ cd static/hello
$ make build
$ make deploy
```

Visit <http://docs.spmjs.org/> for detail information.
